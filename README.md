# 🚀 RAG Series

A collection of **Retrieval-Augmented Generation (RAG)** systems built using different types of input sources such as **URLs, PDFs, Images, and Audio**.

Each project demonstrates how RAG can be applied to extract knowledge from various data formats and answer questions intelligently.

## 🧠 What is RAG?

**Retrieval-Augmented Generation (RAG)** combines:

* 🔍 **Information Retrieval** – Fetching relevant knowledge
* 🤖 **Large Language Models (LLMs)** – Generating intelligent responses

This allows AI systems to answer questions based on **external knowledge sources instead of only pretrained data**.
